# Dakshya
**AI-Powered Career Guidance Platform for Nepali Students**
 
**Dakshya** (दक्ष्य — *efficient* in Nepali), built using Next.js, Node.js/Express, and MongoDB, is a full-stack, AI-powered career intelligence platform designed specifically for Nepali students. It bridges the gap between Nepal's academic system and its job market in one unified platform.
 
Students can receive real-time skill gap analysis against actual Nepali job market demand, generate personalized week-by-week learning roadmaps, practice with an AI-driven interview simulator that scores responses, and discover matched job opportunities from leading Nepali employment sites — all without needing a mentor or living in Kathmandu.
 
Together, these features make Dakshya a complete solution to reduce Nepal's brain drain by showing students the real career opportunities available within the country and equipping them with the precise skills to land those roles.
